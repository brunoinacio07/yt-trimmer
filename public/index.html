Como usar?

1. Insira o ID do vídeo do Youtube no campo Youtube Video ID

2. Com o vídeo carregado, utilize o botão In para selecionar o ponto de início do corte

3. Com o botão Out, selecione o ponto final do corte

4. Clique no botão Play para executar a seleção

5. Caso esteja do seu agrado, clique no botão Download para fazer o download da seleção
